# 菜单图标放置规范

每个菜单项需要两张图片（未选中 + 选中）：

| 文件名               | 说明           |
|----------------------|----------------|
| home.png             | 首页 - 未选中   |
| home-active.png      | 首页 - 选中     |
| class.png            | 班级 - 未选中   |
| class-active.png     | 班级 - 选中     |
| live-class.png       | 实时课堂 - 未选中|
| live-class-active.png| 实时课堂 - 选中  |
| resources.png        | 资料中心 - 未选中|
| resources-active.png | 资料中心 - 选中  |
| message.png          | 消息 - 未选中   |
| message-active.png   | 消息 - 选中     |

建议尺寸：48×48px 或 64×64px，PNG 格式，透明背景。
